# SyntaxWear - Projeto E-commerce

Este é um projeto de e-commerce de tênis e sneakers chamado **SyntaxWear**. O site é uma aplicação front-end estática, construída com HTML e CSS, e foi desenvolvida como parte de um estudo de desenvolvimento web.

## 📖 Visão Geral

O SyntaxWear é uma loja online fictícia que oferece uma variedade de calçados, com foco em tênis e sneakers para diferentes estilos e ocasiões. O layout é moderno e responsivo, proporcionando uma boa experiência de usuário em diferentes dispositivos.

## ✨ Funcionalidades

- **Página Inicial:** Apresenta os produtos em destaque, categorias e promoções.
- **Navegação:** Menu de navegação com links para as principais seções do site, como "Masculino", "Feminino" e "Outlet".
- **Categorias de Produtos:** Seção de categorias que permite aos usuários filtrar os produtos por estilo (Casual, Esporte, Moderno, Futurista).
- **Grid de Produtos:** Exibe os produtos em um layout de grade, com imagens e informações relevantes.
- **Rodapé:** Inclui links úteis, informações de contato, e um formulário de inscrição para a newsletter.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

- **HTML5:** Para a estrutura e semântica do conteúdo.
- **CSS3:** Para a estilização e layout do site.
- **Flexbox e Grid Layout:** Para a criação de layouts responsivos e complexos.

## 📂 Estrutura do Projeto

O projeto está organizado da seguinte forma:

```
/
├── css/
│   ├── base.css
│   ├── reset.css
│   ├── variables.css
│   └── components/
│       ├── footer.css
│       ├── header.css
│       ├── hero.css
│       ├── product-category.css
│       └── product-grid.css
├── images/
│   ├── banners/
│   ├── favicons/
│   ├── icons/
│   ├── logo/
│   └── products/
├── index.html
└── README.md
```

- **`css/`**: Contém os arquivos de estilo do projeto.
  - **`base.css`**: Estilos base do projeto.
  - **`reset.css`**: Reseta os estilos padrão do navegador.
  - **`variables.css`**: Define as variáveis de CSS utilizadas no projeto.
  - **`components/`**: Contém os arquivos de estilo para cada componente do site.
- **`images/`**: Armazena todas as imagens utilizadas no projeto.
- **`index.html`**: O arquivo principal da aplicação, que contém a estrutura da página inicial.
- **`README.md`**: Este arquivo, com a documentação do projeto.
